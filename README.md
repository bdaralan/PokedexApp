# PokedexApp

## iOS Pokedex App

A beautiful Pokedex app for the latest Pokemon including their essential data:

* Stats
* Types
* Abilities
* Moves
* Measurements
* Weaknesses
* Evolutions
* Pokedex Entry.

## Status

* [x] Finished version 1.0.
* [ ] Revamping UI.

## Future Features

* [ ] Mark caught/favorite Pokemon
* [ ] Set reminder to catch Pokemon
* [ ] Create teams
* [ ] Compare teams
